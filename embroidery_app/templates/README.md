# 🌸 Regional Embroidery Simulation System

## Overview
The Regional Embroidery Simulation System is a web-based application that enables users to explore traditional embroidery styles, simulate embroidery designs on fabric, and learn about regional embroidery patterns through an interactive platform.

## Features
- User Authentication
- Dashboard
- Embroidery Design Gallery
- Embroidery Simulation
- Learning Hub
- Favorites / Saved Designs
- Download Design
- Admin Panel

## Technology Stack

## Installation

1. Clone the repository:

```bash
git clone https://github.com/snehapattadkall006-sys/regional-embroidery-simulation-system.git
```

2. Navigate to the project folder:

```bash
cd regional-embroidery-simulation-system
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python app.py
```

5. Open your browser and visit:

```text
http://127.0.0.1:5000
```
